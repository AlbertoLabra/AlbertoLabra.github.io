Lo básico

Lo primero para crear un archivo HTML debemos de abrir nuestros archivos y crear una carpeta con el nombre que queramos y guardamos, luego debemos arrastrar esa carpeta a nuestro área de trabajo y soltamos, posteriormente se abrira la carpeta

Nota :

Al momento de crear un nombre a nuestra página de trabajo le debemos de ponerle un .html al final

Vamos a aprender a crear archivos Html usando la función: HTML:5
Este es el código base del HTML
